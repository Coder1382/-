В REST API приложении реализованы связи "Один ко многим" и "Многие ко многим". Реализованы алгоритмы выполнения следующих запросов:
  • посмотреть все записи / одну запись из таблицы (запрос GET с параметром и без)
  • добавить запись в таблицу (запрос POST)
  • изменить запись в таблице (запрос PUT)
  • удалить запись из таблицы (запрос DELETE)
Даны три таблицы: Fruit, Sellers. Suppliers. Запись в таблице Fruit состоит из двух полей: name и price. У каждой записи предусмотрен список продавцов. Пользователь может изменять цену, добавлять/удалять продавцов в списке.
Запись в таблице Sellers включает одно поле: name. У каждой записи предусмотрен список фруктов, которыми продавец торгует. Пользователь может изменять поставщика для данного продавца, добавлять/удалять фрукты в списке.
Запись в таблице Suppliers включает одно поле: name. У каждой записи предусмотрен список продавцов, с которыми работает поставщик. Пользователь может добавлять/удалять продавцов в списке.
При добавлении, редактировании, удалении записей в таблицах, списки, сопряженные с соответствующими записями, обновляются/удаляются автоматически.

Игра Rogue Game.
Конфигурация следующая: в игре 21 уровень, каждый состоит из 9 комнат, в комнатах могут находиться как враги (вампиры, привидения, огры, зомби, колдуны), так и полезные предметы (золото, эликсиры, свитки, еда, оружие: палка и меч). Комнаты и враги располагаются рандомно. Игрок между комнатами может перемещаться только по определенным коридорам, которые генерируются рандомно, как и двери комнат. Интерьер комнаты виден только когда игрок в нее зашел. В одной из комнат рандомно раполагается портал в следующий уровень.
Цель игры: набрать как можно больше золота и найти портал. С каждым уровнем количество и силы врагов повышаются, а количество полезных предметов уменьшается.
Враги движутся в комнатах по своим паттернам, но с определенного расстояния начинают преследовать игрока. У игрока есть рюкзак, в который можно положить не более 9 предметов.
После каждого уровня прогресс игрока сохраняется в формате JSON
Внизу экрана строка с информацией о состоянии игрока. При приближении врага появляется информация о его способностях.
