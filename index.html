PET-проекты

В Браузерной игре "пилотируемый" треугольник перемещается по экрану между квадратными препятствиями, движущимеся на него. Управление осуществляется стрелками влево/вправо/вперед/назад; ускорение- нажать два раза нужную стрелку. В игре три уровня, отличающиеся частотой появления препятсвий. На третьем уровне препятствия вращаются. Запустить игру можно, перейдя по ссылке. Проект написан на JavaScript, HTML. CSS.
В REST API приложении (JDBC, PostgreSQL) реализованы связи "Один ко многим" и "Многие ко многим". Реализованы алгоритмы выполнения следующих запросов:
  • посмотреть все записи / одну запись из таблицы (запрос GET с параметром и без)
  • добавить запись в таблицу (запрос POST)
  • изменить запись в таблице (запрос PUT)
  • удалить завпись из таблицы (запрос DELETE)
Имеются три таблицы: Fruit, Sellers. Suppliers. Запись в таблице Fruit состоит из двух обязательных полей: название фрукта (name) и цена (price). У каждой записи предусмотрен список продавцов. Пользователь может изменить цену, а также добавлять/удалять продавцов в списке.
Запись в таблице Sellers состоит из одного обязательного поля: имя (name), а также одного опционального: поставщик (supply). У каждой записи предусмотрен список фруктов, которыми продавец торгует. Пользователь может изменить поставщика для данного продавца, а также добавлять/удалять фрукты в списке.
Запись в таблице Suppliers состоит из одного обязательного поля: имя (name). У каждой записи предусмотрен список продавцов, которым поставщик поставляет товар. Пользователь может добавлять/удалять продавцов в списке.
При добавлении, редактировании, удалении записей в таблицах, списки, сопряженные с соответствующими записями, обновляются/удаляются автоматически.
Примеры работы с данным приложением показаны в файле README.md. Проект написан на Java.
Проект REST API приложение (Spring, PostgreSQL) аналогичен предыдущему, но переписан с использованием фреймворка Spring.
Проект ArrayList - моя реализация класса ArrayList, со всеми основными методами, присущими одноименному классу в языке Java.
Linux Socket - это консольное приложение для установления TCP соединения и обмена сообщениями между сервером и клиентами. Сообщения, посылаемые клиентом через консоль, появляются на консоли на стороне сервера. Сервер сохраняет обработанное сообщение в log-файле и возвращает его в консоль клиента.
На стороне сервера через консоль задаётся один параметр: номер порта. На стороне клиента консоль в качестве вводых данных принимает строку, в которой три параметра: имя клиента, номер порта, и время соединения в секундах. Пример использования показан в файле README.md. Проект написан на C++.
